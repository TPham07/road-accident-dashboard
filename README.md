# UK Road Accident Safety Dashboard (2022)

Interactive Tableau dashboard analysing 140,000+ road accidents across the UK.

**Live Interactive Dashboard → [Click here to explore on Tableau Public](https://public.tableau.com/app/profile/thao.pham753/viz/RoadAccidentDashboard_17495862208830/Dashboard1)**

![UK Road Accident Dashboard Preview](dashboard-preview.jpg)

### Key Insights
- 11.7% reduction in total accidents YoY  
- 83.7% of serious casualties occurred in Fine weather  
- Single carriageways account for 78% of serious incidents

### Features
- Year comparison (2021 vs 2022)  
- Breakdown by weather, road type, vehicle type, location  
- Interactive UK map with 27k+ serious casualties  
- Accident severity filter

Built with **Tableau** • UK Department for Transport open data  
Open to **remote junior data analyst / BI roles**
